## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Frontend -> Rust scan_project_icons | 通过 invoke 调用，参数为项目路径 |
| File picker -> Frontend | 系统文件选择器返回本地文件路径 |
| assetProtocol -> WebView | convertFileSrc 将文件路径转为 WebView 可加载 URL |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-05 | Tampering | 文件图标路径存储 | accept | "file:" 前缀图标路径存储在本地 JSON store 中，用户自己选择。不存在网络输入场景 |
| T-08-06 | Information Disclosure | convertFileSrc 暴露本地文件 | accept | 只暴露用户主动选择的图标文件，且 assetProtocol 已在 Plan 01 中配置 scope |
| T-08-07 | Spoofing | 恶意 SVG 文件 | accept | img 标签不执行脚本。convertFileSrc 返回的 URL 作为 img src 不会导致 XSS |

</threat_model>
