## 吸附触发方式

| Option | Description | Selected |
|--------|-------------|----------|
| 松手吸附 | 窗口边缘距屏幕边缘 <= 10px 时松手触发吸附 | ✓ |
| 接触即吸附 | 拖拽进入边缘区域立即触发，无需松手 | |
| 拖入+停留吸附 | 拖到边缘后停留一段时间才触发 | |

**Notes:** 松手吸附符合用户直觉，不会误触。

| Option | Description | Selected |
|--------|-------------|----------|
| 10px 容差 | 窗口边缘距屏幕边缘 <= 10px 时松手触发 | ✓ |
| 5px 容差 | 更精确，避免误触发 | |
| 20px 容差 | 非常宽松 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 四边全支持 | 上/下/左/右均可吸附 | ✓ |
| 仅左右两侧 | 避免 Windows Snap 语义冲突 | |
| 仅右侧 | 最简单 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 滑入动画隐藏 | 松手后窗口平滑滑入边缘隐藏（约 200ms） | ✓ |
| 瞬间隐藏 | 松手后窗口瞬间消失 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 有吸附提示 | 拖拽接近边缘时显示半透明边框提示"即将吸附" | ✓ |
| 无提示 | 无任何视觉反馈 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 禁止吸附 | 最大化状态不触发边缘吸附 | ✓ |
| 允许吸附 | 最大化也可吸附，自动取消最大化 | |

## 滑出与收回行为

| Option | Description | Selected |
|--------|-------------|----------|
| 全窗口滑出 | 鼠标接触细条后窗口完整滑出到吸附前的位置 | ✓ |
| 部分滑出 | 窗口只滑出一部分 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 立即收回 | 鼠标离开窗口后立即收回 | |
| 延迟收回 (300-500ms) | 鼠标离开后等待再收回，期间重新进入则取消 | ✓ |
| 长延迟收回 (1s) | 等待 1 秒 | |

| Option | Description | Selected |
|--------|-------------|----------|
| ease-in-out | 平滑过渡，约 200-300ms | ✓ |
| 弹出式 | 快速弹出 | |
| 弹性动画 | 类似 macOS Dock 弹性效果 | |
