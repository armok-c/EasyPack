    3. `getLinterExtensions(fileName: string): Extension[]` 函数（per D-11, D-12）：
       - `.json` → `[linter(jsonParseLinter(), { delay: 500 }), lintGutter()]`
       - `.xml` → `[linter(xmlLinter, { delay: 500 }), lintGutter()]` 其中 xmlLinter 使用 `new DOMParser().parseFromString()` 检测 parsererror
       - `.yaml`, `.yml` → `[linter(yamlBasicLinter, { delay: 500 }), lintGutter()]` 其中 yamlBasicLinter 做基础格式检查（缩进一致性、冒号分隔符、引号配对）
       - `.toml` → `[linter(tomlBasicLinter, { delay: 500 }), lintGutter()]` 其中 tomlBasicLinter 做基础 key=value 格式检查
       - `.json` 使用 `jsonParseLinter()` 内置函数（per RESEARCH "Don't Hand-Roll"）
       - Lint 函数返回 `Diagnostic[]` 类型（per `@codemirror/lint`）

    4. `formatRelativeTime(timestamp: number): string` 函数（per D-27）：
       - 计算 `Date.now() - timestamp` 差值
       - 返回中文相对时间：「刚刚」「N 分钟前」「N 小时前」「N 天前」
       - 天数精度为整数，向下取整

    5. `ERROR_STATUS_BAR_HEIGHT` 常量用于编辑器底部状态栏布局（值 `"36px"`），此常量供 FileEditorDialog 参考使用。
