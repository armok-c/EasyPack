## Decisions Made

- **Hook API:** Used options object `(parentRef, { value, onChange, darkMode, height, extensions })` instead of flat parameters, consistent with D-37/D-39 from CONTEXT.md
- **Extension merging:** Hook internally merges `basicSetup`, `lineNumbers()`, updateListener, theme, conditional `oneDark`, then appends user extensions -- callers provide only format-specific extensions
- **file-lang.ts structure:** Used `LANGUAGE_MAP` as `Record<string, () => Extension>` factory pattern, enabling lazy initialization of language extensions
- **YAML/TOML linters:** Implemented lightweight format checks (indentation, colon separator, quote pairing) instead of full parsers, consistent with RESEARCH.md Pitfall 4 guidance
