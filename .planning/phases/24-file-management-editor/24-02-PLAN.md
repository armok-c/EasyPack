    1. `addFiles(projectId: string, envId: string, files: ManagedFile[]): Promise<void>`
       - 获取 `projectEnvsMap[projectId]` 当前环境列表
       - 使用不可变更新：找到匹配 `env.id === envId` 的环境
       - 去重逻辑（per D-07）：现有文件名 Set，过滤掉已存在的 `file.name`
       - 更新 `env.updatedAt = Date.now()`，`env.files = [...env.files, ...newFiles]`
       - 若所有文件都是重复（newFiles.length === 0），不执行写入，直接返回
       - 持久化：`profileStore?.set(projectEnvsKey(projectId), updated)` + `await profileStore?.save()`

    2. `deleteFiles(projectId: string, envId: string, fileNames: string[]): Promise<void>`
       - 获取 `projectEnvsMap[projectId]` 当前环境列表
       - 使用不可变更新：找到匹配的环境，创建 `nameSet = new Set(fileNames)`
       - `files: env.files.filter((f) => !nameSet.has(f.name))` 移除匹配文件
       - 更新 `env.updatedAt = Date.now()`
       - 持久化：同上模式

    3. `updateFileContent(projectId: string, envId: string, fileName: string, content: string): Promise<void>`
       - 获取 `projectEnvsMap[projectId]` 当前环境列表
       - 使用不可变更新：找到匹配的环境，在 `env.files` 中找到 `f.name === fileName`
       - 更新匹配文件的 `content` 字段：`{ ...f, content }`
       - 更新 `env.updatedAt = Date.now()`
       - 不触发 toast（per D-16：FileEditorDialog 在保存后自行 toast「已保存」）
       - 持久化：同上模式

    三个方法均使用 `useCallback`，deps 包含 `[projectEnvsMap, profileStore]`。
