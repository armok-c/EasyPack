## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| UI -> Store | 用户选择的图标名和颜色值写入本地 JSON store |
| Store -> UI | 应用启动时从 store 恢复 icon/color 数据渲染到 UI |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | S (Spoofing) | ProjectSettingsDialog | accept | 本地桌面应用，无身份验证需求 |
| T-05-02 | T (Tampering) | updateProjectStyle | mitigate | icon 值必须是 ICON_OPTIONS 中的 key，color 值必须是 COLOR_OPTIONS 中的 value。在 updateProjectStyle 内部验证输入合法性：`if (icon && !(icon in ICON_OPTIONS)) return; if (color && !COLOR_OPTIONS.some(c => c.value === color)) return;` |
| T-05-03 | I (Info Disclosure) | Store JSON | accept | 本地 JSON 文件不含敏感信息（仅项目名称和路径） |
| T-05-04 | D (Denial of Service) | N/A | accept | 本地桌面应用，无服务端 |
| T-05-05 | E (Elevation of Privilege) | Store write | accept | 本地应用已拥有完整文件系统访问权限 |
</threat_model>
