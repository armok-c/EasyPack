<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Tray menu → executeCommand | 用户通过原生菜单触发命令执行，命令字符串来自 store 持久化数据 |
| Tray menu → app.exit(0) | 退出操作无确认弹窗（per UI-SPEC destructive actions 表） |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01 | S (Spoofing) | tray-icon creation | accept | 仅应用自身创建 tray，无外部输入 |
| T-12-02 | T (Tampering) | recentCommands store data | mitigate | store 数据通过 JSON 反序列化验证结构；addRecentCommand 仅接受 name+command 字符串，不执行任意代码 |
| T-12-03 | I (Info Disclosure) | tray tooltip | accept | Tooltip 仅显示 "EasyPack"，无敏感信息 |
| T-12-04 | D (DoS) | tray menu click spam | accept | 命令执行本身已有 toast 反馈，连续点击由用户自行控制 |
| T-12-05 | E (Elevation) | app.exit(0) via tray quit | accept | 退出是预期用户操作，无权限提升风险 |
</threat_model>
