```css

```

Uses OS system font for zero-latency loading in Tauri WebView. Chinese characters render via system fallback (Microsoft YaHei on Windows).

| Element | Copy |
|---------|------|
| Primary CTA | "添加项目" (verb + noun) |

## Layout Specification

### Overall Structure

```
+--[Sidebar 240px]--+--[Main Area flex-1]--------+
|                    |                              |
| EasyPack (title)   |  当前项目: {name}            |
|                    |  路径: {path}                |
| [+ 添加项目]       |                              |
|                    |  +--------+  +--------+     |
| (project name)     |  | 打包   |  | 启动   |     |
|                    |  | 项目   |  | 项目   |     |
|                    |  +--------+  +--------+     |
|                    |  +--------+  +--------+     |
|                    |  | Git    |  | 启动   |     |
|                    |  | Pull   |  | Claude |     |
|                    |  +--------+  +--------+     |
|                    |                              |
+--------------------+------------------------------+
     ^                                              |
     |  720px total (480px height)                  |
```

### Responsive Behavior

| Window Width | Behavior |
|-------------|----------|
| 600-720px (min-default) | Sidebar remains 240px, main area compresses |
| 720px+ | Main area expands, card grid may reflow |
| Below 600px | Not possible (min-width enforced by Tauri) |

### Grid Specification (Command Cards)

| Property | Value |
|----------|-------|
| Columns | 2 (grid-cols-2) |
| Gap | 12px (card-gap token) |
| Card height | auto (content-based, min ~80px) |
| Card padding | 16px (p-4) |
| Card border-radius | 12px (rounded-xl) |
| Card layout | Icon top-center, label below, centered text |
