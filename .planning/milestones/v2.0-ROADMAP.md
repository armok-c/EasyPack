# Milestone v2.0: 能力跃升

**Status:** ✅ SHIPPED 2026-06-12

## Milestone Summary

**Key Decisions:**

- Decision: tauri-plugin-autostart Builder API + --autostart 参数 (Rationale: 官方推荐模式，兼容 Windows 注册表)
- Decision: tempfile::Builder + .keep() 替代 uuid crate (Rationale: 减少依赖，temp 文件管理更安全)
- Decision: ShortcutAction 统一类型覆盖 command/window/project 三类操作 (Rationale: 支持快捷键面板的统一绑定界面)
- Decision: 快捷键 handler 使用 ref 模式避免闭包过期 (Rationale: 解决 React hooks 中 stale closure 问题)
- Decision: 双 store 架构 mainStore + profileStore (Rationale: 分离 profile 元信息和用户数据，支持多配置切换)
- Decision: switchProfile 使用 useRef 并发锁而非 useState (Rationale: 避免 re-render 干扰序列化切换)

**Issues Resolved:**

- 悬浮窗 toggle 并发保护缺失（root_cause_found → 已修复）
- 系统托盘设置冲突（fixed → 静态/动态声明兼容）
- 项目图标扫描（awaiting_human_verify → 已修复）
- WebView2 节流导致窗口操作失败（root_cause_found → 已修复）
- Windows 图标显示异常（verifying → 已修复）

**Technical Debt Incurred:**

- 多行脚本执行结果反馈（成功/失败行数、总耗时）推迟到后续版本
- 配置选择性导入（合并 vs 覆盖）推迟
- CommandDialog.test.tsx pre-existing 测试失败仍未修复
