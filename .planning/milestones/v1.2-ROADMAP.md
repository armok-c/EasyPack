# Milestone v1.2: 快捷键、托盘与窗口增强

**Status:** SHIPPED 2026-05-12
**Phases:** 11-14
**Total Plans:** 11

## Overview

让 EasyPack 从"点击执行"进化为"键盘驱动 + 随手可用"的桌面工具。实现全局快捷键绑定、系统托盘常驻、迷你悬浮窗和边缘抽屉四大功能。

## Milestone Summary

**Key Decisions:**
- 全局快捷键用 tauri-plugin-global-shortcut 而非自建 (Rationale: 官方维护，权限自动管理)
- 托盘菜单显示最近 8 条执行指令 + 项目名 (Rationale: 快速重复执行常用指令)
- TitleBar 齿轮按钮打开统一设置弹窗 (Rationale: 集中管理所有开关设置)
- 悬浮窗使用 Vite 多页构建 + 独立 WebviewWindow (Rationale: 独立窗口生命周期)
- 边缘抽屉用 std::thread 而非 tokio (Rationale: 避免 tokio 直接依赖)
- operationLock Promise-chain mutex (Rationale: 序列化所有窗口动画，防止竞争)
- drawerEnabled 默认 false (Rationale: 用户主动启用，不影响普通用户)
- useRestoreFromDrawer 不改变 visibility state (Rationale: 调用方负责状态管理)
- startDragging() 吞掉 JS 鼠标事件，用 onMoved + debounce 检测拖拽结束 (Rationale: Tauri v2 Windows 限制)

**Issues Resolved:**
- 修复了 useTray/useFloatWindow 中多处 stale closure 问题 (ref 模式)
- 修复了 Switch unchecked 状态视觉对比度不足
- 修复了 closeToTray 时序安全（settingsLoaded 守卫）
- 修复了 FloatApp emit+destroy 竞态条件
- 修复了 useEdgeDrawer 中所有 operationLock 状态突变竞争 (8 轮 code review)
- 修复了 Rust 鼠标轮询 ABBA 死锁风险 (锁序反转)
- 修复了 SnapIndicator 位置计算错误 (fixed positioning 替代 workArea coordinates)

- 8 pre-existing TypeScript build errors
- Phase 11/12 VERIFICATION status: human_needed (OS-level 行为需人工验证)
